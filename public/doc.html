| **Method** | **Endpoint** | **Description** | **Request Body / Params** | **Response** | |------------|-------------------------------------------|-------------------------------------------------------------------|----------------------------------------------------------------------------|------------------------------------------------------------| | `GET` | `/movies` | Get a list of all movies | None | JSON array of all movies | | `GET` | `/movies/:title` | Get movie details by title | `:title` – Movie title | JSON object with movie details | | `GET` | `/movies/genre/:genreName` | Get genre description by genre name | `:genreName` – Genre name | JSON object with genre name and description | | `GET` | `/movies/director/:directorName` | Get director info by director name | `:directorName` – Director name | JSON object with director name, bio, and year of birth | | `POST` | `/users` | Create a new user | JSON body: `{ "name": "User Name" }` | JSON object of the new user | | `PUT` | `/users/:id` | Update a user's name | `:id` – User ID, Body: `{ "name": "New Name" }` | Updated user object | | `POST` | `/users/:id/:movieTitle` | Add a movie to user's favorites | `:id` – User ID, `:movieTitle` – Movie title | Confirmation message | | `DELETE` | `/users/:id/:movieTitle` | Remove a movie from user's favorites | `:id` – User ID, `:movieTitle` – Movie title | Confirmation message | | `DELETE` | `/users/:id` | Delete a user by ID | `:id` – User ID | Confirmation message |
